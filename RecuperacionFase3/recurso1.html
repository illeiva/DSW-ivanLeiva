Etiquetas de estructuracion Semantica

Estas son las principales que debemos definir al diseñar una pagina web es decir cual sera su estructura de bloques, por lo general una pagina web tiene una cabecera de titulo, un contenido principal, etc

  • Este es usado para definir la cabecera de la pagina
  • Este es usado para definir el pie de pagina, este por lo general contiene datos sobre quien es el creador de la pagina o datos de copyright, etc.
  • Este es usado para definir un elemento de navegacion de la pagina web tal como un menu principal o menus secundarios.
  • Este es usado para encerrar el codigo el cual puede corresponder a una seccion generica dentro de un documento.
  • Este es usado para determinar el contenido que no sea principal de la pagina, este se puede usar para todos los elementos secundarios, por ejemplo bloques publicitarios, enlaces externos, etc.
  • Este es usado para definir articulos, este por lo general es el mas importante de HTML5, ya que este permite indicar el contenido de la pagian.
